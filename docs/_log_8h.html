The default error handler will only output the log message if the LOG_SEVERITY <= 1 warn macro function that will write an warning message to the debug output only in the DEBUG build. The default error handler will only output the warning message if the LOG_SEVERITY <= 2 assert macro function that will test the inputed condition and write and error if not true.
